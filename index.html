Les langages informatiques

  • Binaire:

    XVII siècle

    C’est un language informatique qui n’as que deux possibilités, 0 et 1. On en retrouve des traces chez les Indiens et les Chinois, mais l’importance du système binaire a réellement été découverte par Leibnitz.

  • Fortran:

    1954-1957

    Projet lancé en 1954 et aboutissant à une première version en 1957, Fortran est le premier langage de programmation de haut niveau. FORTRAN est utilisé pour des projets tels que la conception de ponts et de structures d'avions, le contrôle de l'automatisation industrielle, ect...

  • Shell:

    fin des années 60

    Un shell est un type de programme informatique appelé interpréteur de ligne de commande qui permet aux utilisateurs de Linux et d'Unix d’entrer directement en communication avec leurs systèmes d'exploitation à l'aide d'interfaces de ligne de commande.

  • C:

    années 70

    Le langage de programmation C est utilisé pour créer une grande variété de logiciels et systèmes d’exploitation. Il est surtout utilisé dans le développement des applications qui nécessitent de hautes performances et des logiciels de base pour les ordinateurs.

  • C++:

    1985

    C++ est un langage de programmation créé par Bjarne Stroustrup, un des langages les plus célèbres au monde. Très utilisé, notamment dans le secteur des jeux vidéo qui apprécie ses performances et ses possibilités.

  • Python:

    1991

    Ce langage a été créé par Guido van Rossum. C’est un langage puissant et polyvalent. Python a de nombreux domaines d'application. Avec Python, on peut créer et administrer un site Web, développer des logiciels et des applications, aussi bien pour ordinateur que pour téléphone.

  • PHP:

    1995

    Le langage PHP a été créé en 1994 par Rasmus Lerdorf pour son usage personnel, principalement utilisé pour produire des pages Web dynamiques via un serveur web. Au fil du temps, PHP est devenu un élément indispensable de nombreux sites Web et technologies Internet modernes, comme Facebook et Wikipédia.

  • JavaScript:

    1995

    Ce langage de programmation, créé par Brendan Eich, doit notamment son succès à son omniprésence dans le développement web avec HTML et CSS JavaScript est un langage qui permet de créer du contenu mis à jour de façon dynamique, de contrôler le contenu multimédia, d'animer des images, ect...

  • Ruby:

    1995

    C’est un langage simple, polyvalent et flexible, beaucoup utilisé dans le développement web. Il a été créé pour combiner les avantages et particularités d’autres langages informatiques, et l’utilisateur peut entre autres modifier le langage lui-même.

  • Java:

    1995

    Créé par James Gosling. Robuste et rapide à l’exécution, Java permet également de faire des jeux-vidéos, et des applications mobile Android. Java a été conçue pour être facile à utiliser et est donc facile à écrire, compiler, et à apprendre. Il vous permet de créer des programmes modulaires et du code réutilisable.

  • C#

    2002

    C# (C sharp) est un langage de programmation commercialisé par Anders Hejlsberg(Microsoft) et destiné à développer sur la plateforme Microsoft.NET. C# est dérivé du C++ et très proche du Java. Il est utilisé pour développer des applications de bureau, des services web, des commandes ou des widgets.

  • Typescript:

    2012

    TypeScript est produit par Microsoft, cette techno est, grossièrement, une surcouche à JavaScript. Elle ajoute à ce dernier plusieurs fonctionnalités attendues par les développeurs : un typage (d’où son nom), des interfaces, la gestion de modules ou encore les paramètres optionnels.
